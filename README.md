## Krishna Health Care 
Integration

## Project Overview
Krishna Health Care is a modern healthcare website built using React (Vite) for the frontend and
Node.js with Express for the backend. The website includes an AI-powered chatbot that can answer
patient queries related to clinic timings, orthopaedic treatments, location details, and appointment
information.

## Key Features
• Responsive healthcare website
• Premium Hero section with dynamic image slider
• Embedded Google Maps location
• Contact section with call and email support
• AI chatbot for instant patient support
• Secure environment variable handling
• Ready for WhatsApp auto-response integration

## Technology Stack
• Frontend: React (Vite), Custom CSS, JavaScript (ES6)
• Backend: Node.js, Express.js, OpenAI API / Free AI alternative
• Deployment: Vercel (Frontend), Render (Backend)

## System Workflow
1. User sends a message through the website chatbot. 2. The message is sent to the backend
server. 3. The backend forwards the message to the AI API. 4. The AI generates a professional
hospital-related response. 5. The response is returned to the user interface.
Setup Instructions
Frontend Setup: - Navigate to healthcare folder - Run: npm install - Start: npm run dev - Build: npm
run build
Backend Setup: - Navigate to backend folder - Run: npm install - Create .env file
